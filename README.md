# Install Guide

* cp .env.sample .env

* Modify .env 

```DB_NAME=${DB}
 DB_USER=${USER}
 DB_PASSWORD=${PASSWORD}
 DB_HOST=${HOST}
 WP_ENV=development
 WP_HOME=${URL}/
 WP_SITEURL=${URL}/wp
 ```
* composer install
